Staff Chat Bungee

There is a need to communicate to staff in any area. Whether it be in a physical business, an online business, or any other organization, communication between staff is a must-have. For the Bungee world, Staff Chat Bungee attempts to facilitate said communication. Let's check it out. Staff Chat Bungee isn't like other proxy staff chat plugins. This plugin paves the way, boasting full message customization, a lean and fast plugin, and active development. We're ready to support your server, to make sure that it makes it to the top. We believe in the "communication is key" philosophy, and this plugin will help you to believe in it as well. Try it now!

Commands and their Permissions

(/command stands for the command you choose.)

Command Permission Description
/command staffchat.toggle Toggle the chat. When toggled, all messages sent are diverted to the staff chat, skipping the public chat.
/command [message] staffchat.send/staffchat.receive Speak to the staff chat with no toggle.
screload staffchat.admin Reload the configuration and language files for the plugin.

Other Permissions

Permission Description
staffchat.colormessage Allows users to color their messages using "&" and the corresponding code.

Installation Instructions

Installing Staff Chat Bungee is extremely easy. Follow these simple steps to have a working installation ready to go:

  1. Download the jar file from this page.
  2. Add the jar file to your Bungee installation's "plugins" folder.
  3. Start the proxy.
  4. Stop the proxy.
  5. Edit the plugin's configuration to your liking.
  6. Edit the Bungee installation's configuration to add permissions to groups, and then users to groups.
  7. Start the proxy.
  8. Use your new staff chat.
